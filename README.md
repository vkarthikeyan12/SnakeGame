# 🐍 Snake Game

A classic snake game built using Python and Pygame.

---
## 📦 Setup

```bash
pip install -r requirements.txt
python main.py
```
---
## 🎮 Controls
- Arrow keys to move the snake.
- Eat apples to grow and score points.
